# Frontend Run Modes

Default:
```bash
flutter run
```

Use Gemini:
```bash
flutter run --dart-define=AI_PROVIDER=gemini --dart-define=API_BASE_URL=http://localhost:8000
```

Use Grok:
```bash
flutter run --dart-define=AI_PROVIDER=grok --dart-define=API_BASE_URL=http://localhost:8000
```

Use Claude:
```bash
flutter run --dart-define=AI_PROVIDER=claude --dart-define=API_BASE_URL=http://localhost:8000
```
